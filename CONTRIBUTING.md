# How to Contribute to NebulaAPI

Thank you for helping grow this project! Forks and PRs are welcome to snowball NebulaAPI.

## Steps to Contribute
1. Fork the repo.
2. Clone the fork: `git clone https://github.com/aegisflow/nebulaapi.git`.
3. Create a branch: `git checkout -b my-feature`.
4. Test locally: `npx wrangler dev`.
5. Commit: `git add . && git commit -m "Added feature X"`.
6. Push: `git push origin my-feature`.
7. Open a PR: GitHub > New pull request > Compare branch.

## Rules
- Focus on improvements (docs, tests, basic features).
- No resale or commercial use without premium license.
- Review with `wrangler check`.
